# Meyou Sunflower Picnic Assets

이 ZIP은 ChatGPT에서 생성한 이미지를 웹 프로젝트용 폴더 구조와 영문 파일명으로 정리한 패키지입니다.

## 사용법

1. 압축을 풉니다.
2. 안의 `assets` 폴더를 프로젝트 루트의 `assets` 폴더와 병합합니다.
3. CSS/HTML에서는 `/assets/common/...`, `/assets/ladder/...` 같은 경로로 사용합니다.

## 주의

- 선택 카드 문구는 이미지에 박지 말고 HTML/CSS 텍스트로 올리는 것을 권장합니다.
- 일부 파일은 생성 이미지 특성상 실제 투명 PNG가 아니라 체크무늬 배경이 포함되어 있을 수 있습니다. 적용 전 어두운 배경에서 확인하세요.
- `asset-manifest.csv`에 원본 한글 파일명과 최종 경로를 같이 적어두었습니다.
