범례 · 참고사항
투명 배경(PNG) 여부 판단 기준
• 페이지 배경(#080610) 위에 떠있어야 하는 인물·장식 → 필요
• CSS border-radius:50% 원형 클립만 쓰는 경우 → 불필요
• object-fit:cover로 채워지는 배경형 → 불필요
파일 포맷 선택 가이드
• 투명배경 인물·장식 → PNG-24
• 배경없는 로고·아이콘 → SVG
• 배경 있는 사진·카드 → WebP (JPG 대체)
• 애니메이션 데코 → WebP/APNG
현재 base64 이미지 교체 방법
reaction.html 내 src="data:image/png;base64,..."를
src="/assets/reaction/파일명.png" 로 교체만 하면 됨.
HTML 구조·CSS 변경 불필요.
